# Amanda Lee | Portfolio

A personal portfolio site built as part of a career transition from clinical social work into tech.

After 5+ years as an LCSW, I'm pivoting into front-end development and technical customer success. This site was built from scratch as a hands-on learning project.

## Built With

- HTML
- CSS (custom properties, flexbox, grid, responsive design)
- JavaScript (form validation)
- Hosted via GitHub Pages
